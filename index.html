Java CLI

Software Engineering Project: NUS Classes

NUS Classes is a Java Command Line Interface based application developed for a software engineering project. It allows users (professors) to talk to a responsive chatbot to schedule their classes, manage contacts of TAs and students as well as meetings. It is developed using Java, JavaFX and JUnit tests. The project involves software engineering concepts such as automated testing, JUnit tests, as well as using learnt software engineering concepts and paradigms such as AGILE, UML Diagrams, detailed User Guide & Developer Guide documentation, identifying user needs, proper design architecture and more.

Full-Stack

Independent Software Development Project: Notable

Notable is a note sharing application developed with the intention of allowing students to contribute their notes for a module once they have finished taking it. Juniors that are taking the modules in future can download the notes uploaded by their seniors, which can be official lecture notes or self created ones, with ratings for notes & profile pages for users.

Notable implements CRUD (Create, Read, Update, Delete) operations, with its front end in TypeScript, JavaScript & React framework and the backend being Google Firebase & Google Cloud Firestore.
